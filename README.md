# Aptos Defender Demo

## Installation

1. Install python 3.10 or higher

2. Create virtual environment

   ```bash
   python -m venv venv
   ```

3. Activate virtual environment

   ```bash
   source venv/bin/activate
   ```

4. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```

## Run

```bash
python main.py
```
